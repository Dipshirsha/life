Project Name
Description
This project is built using EJS, Node.js, Express, and MongoDB. It provides a dynamic web application with server-side rendering using EJS templates.

Installation
Clone the repository:

bash
git clone <repo_url>
cd <project_directory>
Install dependencies:

bash
npm i
Usage
Start the server:

bash

node app.js
Open your browser and visit:

a

Technologies Used
EJS – For templating
Node.js – Backend runtime
Express – Web framework
MongoDB – Database
